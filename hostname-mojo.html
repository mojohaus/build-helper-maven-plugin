build-helper:hostname

Full name:

org.codehaus.mojo:build-helper-maven-plugin:3.6.1:hostname

Description:

Retrieve current hostname and place it under a configurable project property

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 3.0.0.
  • Binds by default to the lifecycle phase: process-test-classes.

Optional Parameters

Name Type Since Description
<hostnameProperty> String - The name of the property in which to store the hostname.
Default: hostname

Parameter Details

<hostnameProperty>

The name of the property in which to store the hostname.
  • Type: java.lang.String
  • Required: No
  • Default: hostname
