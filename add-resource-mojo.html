build-helper:add-resource

Full name:

org.codehaus.mojo:build-helper-maven-plugin:3.6.0:add-resource

Description:

Add more resource directories to the POM.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.3.
  • Binds by default to the lifecycle phase: generate-resources.

Required Parameters

Name Type Since Description
<resources> Resource[] - Additional resource directories.

Optional Parameters

Name Type Since Description
<skipAddResource> boolean 3.5.0 Skip plugin execution.
Default: false
User Property: buildhelper.addresource.skip
<skipAddResourceIfMissing> boolean 3.5.0 If a resource directory does not exist, do not add it as a root.
Default: false
User Property: buildhelper.addresource.skipIfMissing

Parameter Details

<resources>

Additional resource directories.
  • Type: org.apache.maven.model.Resource[]
  • Required: Yes

<skipAddResource>

Skip plugin execution.
  • Type: boolean
  • Since: 3.5.0
  • Required: No
  • User Property: buildhelper.addresource.skip
  • Default: false

<skipAddResourceIfMissing>

If a resource directory does not exist, do not add it as a root.
  • Type: boolean
  • Since: 3.5.0
  • Required: No
  • User Property: buildhelper.addresource.skipIfMissing
  • Default: false
