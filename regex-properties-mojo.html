build-helper:regex-properties

Full name:

org.codehaus.mojo:build-helper-maven-plugin:1.11:regex-properties

Description:

Sets a property by applying a regex replacement rule to a supplied value. This is similar to regex-property goal with support for multiple regex settings using RegexPropertyConfig

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 1.9.
  • Binds by default to the lifecycle phase: validate.

Optional Parameters

Name Type Since Description
regexPropertySettings List 1.9 List of RegexPropertyConfig to apply the regex

Parameter Details

regexPropertySettings:

List of RegexPropertyConfig to apply the regex
  • Type: java.util.List
  • Since: 1.9
  • Required: No
